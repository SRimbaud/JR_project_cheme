long int  v,
int  mode 
)

Enregistre dans a l'entier long v, retourne le nouveau num. Met à jour champ numtype à NUM_INTEGER.

Paramètres
num*: num ou on doit enregistrer la valeur.
v: entier long à enregistrer.
mode: Mode NUM_PINFTY NUM_MINFTY ou NUM_INTEGER

Si le mode n'est pas une constante implémentée dans la fonction :

  • NUM_REAL (Not implemented)
  • NUM_INTEGER
  • NUM_UINTEGER (Not implemented)
  • NUM_COMPLEX (Not implemented)
  • NUM_UNDEF
  • NUM_PINFTY
  • NUM_MINFTY Il est mis par défaut à NUM_UNDEF
    Renvoie
    : Renvoie un pointeur sur le num modifié.
